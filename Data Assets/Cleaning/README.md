# Cleaning
This folder contains all the scripts used to clean the raw data.
## File Information 
* `Data_Cleaning.Rmd`: R Markdown file for cleaning wind, oil, gas data.  
* `Drough_WHP_Cleaning.Rmd`: R Markdown file for cleaning drought and wildfire risk data. 
* `final_combine.ipynb`: Jupyter notebook for combining all 5 metrics with a county shapefile.
* `cleaner.ipynb`: Jupyter notebook for combining the state shapefile with the dataset. Also applies the Pareto logic to the data.
