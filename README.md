# D2I-Challenge
The official GitHub repository for team Cloud Chaser's D21 Challenge submission - a Data Center Environmental Siting Optimizer.

### Data Assets
The raw data sources, scripts used to clean and combine them, and the final curated data set.

### Decision Tool
Our chosen decision tool, an interactive Data Center Siting Optimizer web app, is deployed from this folder. It contains the dataset, python script, and link to the app.

### Impact Note
A report on explaining our stakeholders, decisions, key assumptions/uncertainties, and plans for future work.

### Presentation
A comphrehensive presentation outlining the problem statement, value proposition, conceptual solution overview, key insights, ethical considerations, and future work of our Data Center Siting Optimizer.
